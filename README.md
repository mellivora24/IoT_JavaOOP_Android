* Cú pháp git tạo nhánh mới:

# Quy trình commit code:
* Chuyển sang nhánh của anh em (nếu trước đó đang ở nhánh main)
~~~ bash
git checkout <tên nhánh AE đã tạo>
~~~
* Commit code:
~~~ bash
git add .
git commit -m "<message rõ ràng>"
git push
~~~
